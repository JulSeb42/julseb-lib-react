# CLI and Boilerplates

To ease your development process, you can use our CLI to generate a boilerplate for React apps. There you can find:

- <a href="https://github.com/JulSeb42/julseb-lib-boilerplate-fullstack" target="_blank" rel="noreferrer noopener">React and Rest API</a>
- <a href="https://github.com/JulSeb42/julseb-lib-boilerplate-client" target="_blank" rel="noreferrer noopener">React (client only)</a>
- <a href="https://github.com/JulSeb42/julseb-lib-express-rest" target="_blank" rel="noreferrer noopener">Server app (in Express)</a>

To use the CLI, open your terminal and run:

```shell
$ npx @julseb-lib/julseb-cli@latest
```

Thanks to the CLI, you will have a brand new app in React without having to build everything by yourself.

Each boilerplate is generated with Plop.js. To help you personalise generated code, with each boilerplate you can find a plopfile with different generators.

## React and Rest API

This boilerplate comes with built in auth, admin panel, protected and anonymous routes, user roles, etc. It uses MongoDB as database.
